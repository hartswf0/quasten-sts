1. Problem Statement

  1. Manovich, Lev - The Language of New Media (2001)

Focused Problem Statement: Describing generative AI art tools like Midjourney (L1 Machine Layer) simply by their outputs (images) obscures their underlying operational logic, modeled by Manovich (L3 Framing) through principles like automation and variability. Analyzing user 'promptcraft' (L2) reveals interaction with this computational logic (L4 Latent Force), which redefines image ontology (L0 Reality) beyond traditional media categories.

2. Core Model

27. Manovich, Lev - The Language of New Media (2001)

  • MODEL: Identifies key principles that distinguish new media (computer-based media) from old media, arguing that its "language" is based on concepts derived from computing: numerical representation, modularity, automation, variability, and transcoding (the translation of cultural categories into computational ones).
  • Index:
    • ◻ New Media (Computer-Based) ←→ ◻ Old Media (Distinguished from)
    • ◻ Computing Concepts → ◻ Language of New Media
    • ◻ Language of New Media → ◻ Principles (Numerical Representation, Modularity, Automation, Variability, Transcoding)
    • ◻ Transcoding → ◻ Translation of Culture → ◻ Into Computational Form

4. Spatiotemporal Framework

27. Manovich, Lev - The Language of New Media (2001)

  • ◻ SPATIOTEMPORAL UNIVERSE: Culture in the age of computers and digital information.
    • → [Contains] → ◻ FIELD: The forms, aesthetics, and underlying principles of "new media" (computer-based media).
      • → [Has] → ◻ DOMAIN: Digital artifacts (websites, interfaces, games, databases, interactive installations), software tools, computer hardware logic × Historical context of computing and modern art/media.
      • → [Projects] → ◻ SIGNAL: Interface designs, data structures, algorithmic processes, interactive behaviors, visual aesthetics of digital works. (ℝᵏ representing resolution, code complexity, interactivity options, database structure).
      • → [Admits] → ◻ PERTURBATION: Development of new software/hardware, changing internet architectures, evolving aesthetic conventions, user innovations.
    • → [Observed Through] → ◻ WINDOW: Comparative media analysis (film theory, art history), software studies, computer science principles, cultural history.
      • → [Defines] → ◻ OBSERVATION SCALE: Focus on identifying the fundamental, recurring principles that distinguish new media from old media, stemming from its computational basis.
      • → [Delimits] → ◻ CONTEXT BOUNDARY: Primarily examines the logic inherited from computer science and its manifestation in cultural forms, linking back to histories of modernism and avant-garde film.
      • → [Generates] → ◻ SAMPLING LATTICE: Analysis of websites, CD-ROMs, video games, digital cinema, user interfaces, database art, software like Photoshop/After Effects.
    • → [Manifests As] → ◻ PHENOMENON: The Language of New Media – Principles derived from computer science that characterize new media objects and processes: Numerical Representation, Modularity, Automation, Variability, Transcoding. These principles shape the aesthetics and possibilities of new media culture. The database vs. narrative is a key resulting tension.
  • ◻ ENGINE: Principle Identification / Comparative Media Formalism.
    • → [Applies] → ◻ TRANSFORM: Analyzing diverse new media objects to identify underlying shared principles stemming from their computational nature and relating them to older media logics.
      • → [Operates On] → ◻ PATCH: Specific new media artifacts, software functions, interface elements.
      • → [Produces] → ◻ INVARIANT: The five principles (Numerical Representation, Modularity, Automation, Variability, Transcoding); the database/narrative dichotomy.
      • → [Requires] → ◻ CONTRACTION: Abstracting from specific content to the underlying formal and structural logic derived from computation.
    • → [Constructs] → ◻ REPRESENTATION: A systematic framework ("language") for understanding the unique characteristics of new media.
    • → [Optimizes] → ◻ RECURSION: Applying the five principles and database/narrative logic to interpret a wide range of new media phenomena.
      • → [Embeds] → ◻ MODEL: Computer science fundamentals, film theory (esp. Vertov), art history (modernism), media theory.
      • → [Projects] → ◻ METRIC: Presence/prominence of the five principles, relationship to database/narrative forms (qualitative/structural).
      • → [Seeks] → ◻ FIXED POINT: Establishing a coherent theoretical foundation for understanding new media based on its computational logic.
  • ◻ METAMODEL: Theory of New Media Principles. Governs the analysis of digital culture based on computational logic. Binds Computer Science Principles/Logics DOMAIN ↔ New Media Forms/Aesthetics/Culture CODMAIN.
  • Distinctions & Crossings:
    • ◻ MODEL (Manovich's 5 principles) | ◻ REALITY (Diverse and rapidly changing new media practices) → ◻ MEASUREMENT ARTIFACT (The principles are a powerful model but might create artifacts by over-emphasizing computational logic or downplaying other factors).
    • ◻ TIME-LIKE (Narrative form, automation processes) | ◻ SPACE-LIKE (Database structure, modularity, interface layout). Explores the tension and interplay between time-based narratives and space-like database logics as central to new media.
    • ◻ CONTINUOUS (Analog reality, cultural forms) | ◻ DISCRETE (Numerical representation, pixels, database entries, modules) → ◻ SAMPLING THRESHOLD (Numerical representation and modularity are foundational discrete thresholds enabling new media). Transcoding is the crucial crossing point.
  • Operations: Observe new media objects and software. Encode them according to the five principles. Convolve older media forms (PATCH) with computational logic (KERNEL) to produce new media FEATUREs (e.g., variability). Entangle computer logic (FEATURE) with cultural forms (FEATURE). Tries to Forget seeing new media as simply faster/better old media.
